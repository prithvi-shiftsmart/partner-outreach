# State: dormant

## Goal
Warm re-engagement. Surface what's changed since they went quiet. One message — if no response, stop. Human decides next steps.

## Available Actions
- Send one re-engagement message with a different angle than the last
- Surface new shifts, orientation slots, or bonuses that weren't available before
- Transition back to previous active state if partner responds

## Context to Inject
- Last active state before dormancy
- Time since last message
- What was the last topic discussed
- Any new orientation options or shifts since they went quiet
- Number of unreturned messages already sent

## Response Guidelines
- Different angle from the last message — if we led with orientation, lead with pay this time
- Keep it short: 1-2 sentences max
- Don't guilt trip ("We haven't heard from you")
- Frame around what's new: "A few new shifts opened up near you" or "Just wanted to check if you had any questions"
- After 2 unreturned messages total: STOP. Flag for ops review. No more automated outreach.

## Transition Triggers
- Partner responds → return to `answering_qs`
- 2 unreturned messages sent → mark as `do_not_auto_message`, requires human decision
