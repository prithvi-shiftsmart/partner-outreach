# Orientation Process — Source of Truth

## Remote Orientation
- 7 modules completed in the app (async, self-paced)
- Takes ~30-45 minutes total
- Partner gets paid $10 for completing it
- Available in remote-enabled priority markets
- Program launch date: 2026-02-19
- Data source: `shiftsmart-api.shiftsmart_data.bq_learning_module_results`

## In-Person Orientation Types
- **Shadow orientation**: one-on-one with experienced M1 partner. Partner gets paid full shift rate.
- **Ambassador-led / Modular**: classroom-style group orientation.

## Funnel Steps
1. Partner downloads app (DL)
2. Completes BGC + profile setup
3. Orientation assigned (OA) — remote or in-person
4. Orientation passed (OP) — completes all modules or attends in-person
5. Eligible for shifts

## Common Drop-Off Points
- Module 3-4 of remote orientation (content gets more detailed)
- Can't find orientation tab in the app
- No in-person slots available near them
- BGC not yet completed (blocks orientation access)

## What Counts as Completed
- Remote: all 7 modules status = 'completed' AND remote auto-creation shift completed
- In-person: assignment_status = 'Completed' on orientation shift type
